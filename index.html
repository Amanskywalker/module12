Module 12

Module 12, I know that a weird name. But this project started as the to implement lexer in C suitable for a simple command language that handles commands, numbers, strings, and new lines, ignoring white space and comment. and it was the 12th module in the series which i picked to do.

Implementation

It uses the regular expression to match the user input specified in the grammar ( I tried to keep grammar as closed to the original lexical analyser). A Few new custom function are made like preg_match() to implement the same functionality which is there in PHP like languages. It also uses the clean coding method to separate the declaration part from the implementation so it easy to further extend. and also make it easy to understand. It used a new size less custom data type vector. Trying to implement c++ vector in c.

Instruction

  • download or clone the repo
  • navigate to root directory
  • use make to build the binary
  • $ ./mod12 to run it
